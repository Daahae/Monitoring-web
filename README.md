# SW모니터
__세종 해커톤 - SW역량 모니터링 부문__

## 개요
SW교과목의 인증시험(C프로그래밍)과 SW경시대회 등의 데이터를 활용하고 분석하여 학생들의 SW역량을 모니터링 할 수 있는 시스템을 구현한다.
